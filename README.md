# Dice Battle Challenge 🎲

## Project Overview
The *Dice Battle Challenge* is a dynamic web application that simulates a competitive dice-rolling game. It uses randomized logic to determine a winner between two players, providing instant visual feedback and score declarations.

## Why I Built This
This project was a practical exercise in *JavaScript Math logic* and *DOM manipulation*. I focused on creating a seamless user experience where the UI updates instantly based on backend logic, ensuring that the game feels "live" and responsive.

## Key Features
* *Randomization Engine:* Leverages Math.random() and Math.floor() to generate unpredictable game outcomes every time the page refreshes.
* *Dynamic Image Swapping:* Uses JavaScript to manipulate the src attribute of image elements, matching the dice face to the generated random number.
* *Smart Winner Declaration:* Implemented conditional logic to compare results and update the <h1> element with the winner's name or a "Draw" status.
* *Responsive Design:* Styled with CSS to ensure the battle layout looks great on both mobile and desktop screens.

## Technical Skills Applied
* *JavaScript:* Logic implementation, random number generation, and DOM selection.
* *CSS3:* Flexbox for layout alignment and responsive design.
* *HTML5:* Semantic structure for a clean, accessible gaming interface.
